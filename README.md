# mini-project-1-1
일만 시간의 법칙(반응형 페이지 구현)
어떤 분야의 전문가가 되기 위해 필요한 '1만 시간'을 달성하기까지 며칠이 걸리는지 계산해 주는 웹 서비스입니다.

📌 프로젝트 개요
목표: 사용자가 설정한 목표와 일일 훈련 시간을 바탕으로 1만 시간을 채우는 데 필요한 기간을 시각화하여 동기부여를 제공합니다.

핵심 문구: "연습은 어제의 당신보다 당신을 더 낫게 만든다."

🚀 주요 기능
1. 시간 계산기 (Calculator)
목표 설정: 전문가가 되고 싶은 분야(예: 프로그래밍)를 입력합니다.

시간 입력: 하루에 투자할 수 있는 훈련 시간을 입력합니다.

결과 출력: 1만 시간 달성을 위해 필요한 총 일수(Day)를 계산하여 화면에 표시합니다.

2. 동기부여 및 공유
훈련하러 가기: 실제 행동으로 옮길 수 있도록 독려하는 버튼을 제공합니다.

공유하기: 결과나 서비스를 타인과 공유할 수 있는 기능을 포함합니다.

3. 응원 오버레이 (Overlay)
사용자의 꿈을 응원하는 캐릭터(라이켓)와 응원 메시지가 담긴 팝업/오버레이 화면을 구현하였습니다.

🛠 기술 스택
HTML5: 시멘틱 태그를 활용한 웹 구조 설계 및 input, button 등의 폼 요소 활용

CSS3: 레이아웃 구성 및 웹 폰트, 이미지 배치 (외부 스타일시트 style.css 연결)

Web Standard: 웹 접근성을 고려한 aria-label 속성 적용
